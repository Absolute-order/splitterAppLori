# レシートスプリッター プレゼン用スピーキングノート 📋

## ⏱️ 想定時間: 20-25分

---

## パート1: 導入 (1分)

**言う:**
```
皆さん、こんにちは！今日は「レシートスプリッター」という新しいアプリを紹介します。

このアプリは、レストランやカフェで、友達とお金を簡単に分けるために作りました。

5人で食べても、1人で食べても、正しい金額を自動で計算してくれます。

では、新しい機能を一つずつ見ていきましょう。
```

**やること:**
- スマートフォンを手に取る
- ホーム画面を表示する

---

## パート2: 友達管理 (2分)

**言う:**
```
まず、「友達管理」という機能です。

ここから、友達を追加、検索、削除できます。
```

**やること:**
1. 左メニュー → 「友達」をタップ
2. 友達一覧を指で示す
3. 右下の🟢プラスボタンを指で示す
4. 「検索画面に移ります」と言ってから、進む

**言う:**
```
友達を追加するには、ここでユーザーIDか名前を入力します。
すると、相手にリクエストが送られます。
相手が承認すると、友達になります。

友達を削除したいときは、ここをタップして、削除ボタンを押します。
```

---

## パート3: リクエスト通知 (1分30秒)

**言う:**
```
次に、「リクエスト通知」です。

右上のベルのマークを見てください。
ここに、友達からのリクエストが表示されます。
```

**やること:**
1. ベルアイコンを指で示す
2. タップして、リクエスト画面を表示

**言う:**
```
ここに、2つのリストがあります。

1つ目は「もらったリクエスト」です。
他の人が、あなたを友達に追加したい、ということです。
「承認」ボタンで友達になります。
「拒否」ボタンで断ることもできます。

2つ目は「送ったリクエスト」です。
あなたが友達を追加したけど、まだ相手が承認していない状態です。

リクエストを承認すると、2.5秒後に自動的に消えます。
これで、画面がすっきり見えます。
```

---

## パート4: レシートスキャン (2分)

**言う:**
```
ここから、メイン機能の「レシートスキャン」です。

ホーム画面に戻ります。
```

**やること:**
1. ホーム画面に戻る
2. 左上の🟢「スキャン」ボタンを指で示す

**言う:**
```
このボタンを押すと、カメラが開きます。
レシートをカメラで撮ります。

アプリが、Google Vision というAI技術を使って、
自動的に品目を読み込みます。

商品名、価格、数量が見つかります。
ほぼ正確ですが、もし間違っていたら、次の画面で修正できます。
```

---

## パート5: 品目の編集 (2分)

**言う:**
```
スキャン後、品目の一覧が表示されます。

ここで、3つのことができます。
```

**やること:**
1. 品目一覧を指で示す
2. 各ボタンを1つずつ指で示しながら説明

**言う:**
```
1つ目は「品目の編集」です。
品名、価格、数量が違う場合は、ここで修正します。

2つ目は「削除」です。
スキャン時に余計な文字が読み込まれた場合、削除します。

3つ目は「追加」です。
読み込み漏れがあった場合、ここで追加します。

すべて確認できたら、「続ける」ボタンを押します。
```

---

## パート6: 参加者と品目の割り当て (2分30秒)

**言う:**
```
次に、「誰が何を食べたのかを決める」という作業です。

画面に、参加者の名前が表示されます。

1つ目の品目を見てください。
下に、参加者のボタンが並んでいます。

もし、A さんがこの品目を食べたなら、A さんのボタンをタップします。
```

**やること:**
1. 品目を1つ指で示す
2. 参加者ボタンを1つタップする
3. 「こうやって、品目が割り当てられます」と言う

**言う:**
```
この作業を、すべての品目が終わるまで繰り返します。

もし、1つの品目を複数の人で分けたい場合は、
何人かを選ぶことができます。

すべて割り当てたら、「確定」ボタンを押します。
```

---

## パート7: 自動計算と結果 (1分30秒)

**言う:**
```
これが、レシートスプリッターのすごいところです！

ボタンを押すと、
自動的に、誰が誰にいくら払うべきか、が計算されます。

複雑な計算をアプリが1秒でやってくれます。
```

**やること:**
1. 結果画面を表示
2. 支払い額を指で示す
3. 「A さんは B さんに 800円、B さんは C さんに 500円...」と説明

---

## パート8: シェア機能 (1分)

**言う:**
```
計算が終わったら、友達と結果をシェアしたいですね。

「共有」ボタンを押すと、
LINE、メッセージ、メール、その他のアプリで送信できます。
```

**やること:**
1. 「共有」ボタンを指で示す
2. タップして、シェアオプションを表示

---

## パート9: 支払い記録と通知 (1分30秒)

**言う:**
```
支払いが終わった後、アプリで記録することができます。

「支払い済みにする」ボタンを押すと、
緑のチェックマークが表示されます。

また、支払う義務がある場合は、
ベルアイコンにオレンジ色の通知が表示されます。

これで、支払い漏れを防ぐことができます。
```

---

## パート10: グループ機能 (2分)

**言う:**
```
次に、「グループ機能」です。

いつも同じメンバーで食事するグループありますね。
野球部とか、家族とか。

レシートスプリッターなら、グループを作成できます。
```

**やること:**
1. グループ画面を表示
2. 「新しいグループを作成」ボタンを指で示す

**言う:**
```
グループの名前を入力して、メンバーをすべて選びます。

すると、同じメンバーで食事をするときは、
グループから「このグループで割り勘を始める」を選ぶだけです。

メンバーが自動的に選ばれているので、
毎回メンバーを選ぶ手間がなくなります。
```

---

## パート11: 履歴とエクスポート (1分30秒)

**言う:**
```
過去の食事会の記録を見たいときは、「履歴」画面を開きます。

ここに、すべての食事会が一覧表示されます。
日付、参加者、合計金額が分かります。

1つの食事会をタップすると、詳細が見えます。
品目、支払い額、支払い状況が表示されます。

また、「もう一度」ボタンを押すと、
同じメンバー、同じ品目で新しい食事会を始めることができます。
```

**やること:**
1. 履歴一覧を表示
2. 1つの明細をタップして、詳細を見せる

**言う:**
```
さらに、右上のダウンロードボタンで、
すべての履歴をファイルで出力できます。

Excel や CSV で保存できるので、
家計管理に使ったり、統計を見たりできます。
```

その他にも、便利な機能がたくさんあります。

1つ目は「多言語対応」です。
日本語、英語、ウズベク語に対応しています。
設定から言語を変えることができます。

プロフィール写真を設定すると、友達一覧で分かりやすくなります。
```

**やること:**
1. 設定画面を表示
2. 言語選択オプションを指で示す
3. プロフィール画面を表示

**言う:**
```
では、新しい機能をまとめます。

✅ 友達管理 - 友達を追加、検索、削除

✅ シェア機能 - 結果を友達にすぐ送信
✅ 支払い記録 - 支払い状況をアプリで管理
✅ 支払い通知 - 支払い漏れを防ぐ
✅ グループ機能 - よく一緒の人をグループ化
✅ 履歴管理 - 過去の食事会を保存・確認
✅ エクスポート - データをファイルで出力
✅ 多言語対応 - 日本語、英語、ウズベク語
✅ プロフィール - ユーザー情報を管理

このアプリを使うことで、

## 💡 質問への返答例

**Q: 「操作は難しくないですか？」**
```
いいえ、とても簡単です。
タップして、ボタンを押すだけです。
高齢の親も使っています。
```

**Q: 「本当に正確に計算されますか？」**
```
はい、正確です。
アプリが自動で計算するので、
人間の手計算より間違いが少ないです。
```

**Q: 「インターネット接続が必要ですか？」**
```
基本的には、はい。
ただし、情報をアップロードした後は、
オフラインでも見ることができます。
```

**Q: 「データは安全ですか？」**
```
はい、暗号化されて、安全に保存されます。
個人情報も保護されています。
```

**Q: 「いくらですか？」**
```
基本は無料です。
将来、プレミアム機能が追加される予定ですが、
今は完全に無料で使えます。
```

## 🎬 デモンストレーション チェックリスト

プレゼン前に、以下をすべてチェックしてください：

- [ ] スマートフォンは充電されているか
- [ ] Wi-Fi または 4G に接続しているか
- [ ] アプリはインストール・起動されているか
- [ ] サンプルの友達が登録されているか
- [ ] サンプルのリクエストがあるか
- [ ] スキャン済みのレシートがあるか
- [ ] グループが作成されているか
- [ ] 履歴データが存在するか
- [ ] 画面は明るく設定されているか（見やすいように）
- [ ] マイクの音量は大きいか

## 🌟 プレゼンのコツ

### 声：
- ゆっくり、はっきり話す
- 難しい単語のときは、いったん止めて、説明を加える

### ジェスチャー：
- スマートフォン画面を聴衆に見せながら操作する
- 「ここをタップします」と言ってから、実際にタップする
- 指で画面要素を示す

### リズム：
- 1つの機能に 2-3分くらい使う
- 途中で「何か質問ありますか？」と聞く
- テンポよく進める

### 親戚性：
- 「これは、私たちが実際に使った時の体験です」のような実例を交える
- 相手の経験に結びつける
