# レシートスプリッター — 使い方（日本向け）

## 基本の流れ

**レシート → 割り勘 → 共有 → 精算**

1. ホームで **スキャン** または **手入力**
2. **品目を編集**（スキャン後）
3. 各品目で **誰？** → **続ける**
4. サマリーで **共有**
5. 通知で **未払い** を確認
6. 履歴で **支払い済み** を記録
7. **もう一度** で同じメンバー・品目から再開
8. **グループ** からメンバー入りで開始

---

## 1. レシートを登録

| 場所 | 操作 |
|------|------|
| ホーム | 緑の **スキャン** または **手入力** |

---

## 2. 品目を編集（スキャン後・中優先度）

| 場所 | 操作 |
|------|------|
| 分割画面（品目一覧） | 上部の緑枠「品目を確認」→ 各品目の **編集** |
| モーダル | 品名・単価・数量を修正、不要な品目は **削除** |

---

## 3. 割り当てと確定

| 場所 | 操作 |
|------|------|
| 分割画面 | 各品目 **誰？** → 参加者を選択 |
| 下部 | すべて割り当て後 **続ける** |
| サマリー | 一人あたりの金額を確認 |

---

## 4. 結果を共有（高優先度）

| 場所 | 操作 |
|------|------|
| サマリー画面 | **共有** → LINE 等にテキスト送信 |
| 履歴 → 明細詳細 | 同じ **共有** ボタン |

---

## 5. 未払いのお知らせ（高優先度）

| 場所 | 操作 |
|------|------|
| ヘッダー 🔔 | 確定後、支払額の通知（オレンジの財布アイコン） |
| タップ | 該当の明細へ |

---

## 6. 支払い済みの記録（高優先度）

| 場所 | 操作 |
|------|------|
| サマリー | **支払い状況 →** で履歴へ |
| 履歴 → 明細詳細 | 各参加者カードの **支払い済みにする** |
| 記録後 | 緑枠・チェック表示。作成者に通知 |

---

## 7. もう一度割り勘（高優先度）

| 場所 | 操作 |
|------|------|
| 履歴 → 明細詳細 | 緑の **もう一度** |
| 分割画面 | 同じ品目・参加者で再割り当て |

---

## 8. グループで開始（高優先度・テンプレート）

| 場所 | 操作 |
|------|------|
| グループ詳細 | **グループで割り勘を始める** |
| 次 | スキャンまたは手入力（メンバーは事前入力済み） |

---

## 9. 履歴のエクスポート（ファイル送信）

| 場所 | 操作 |
|------|------|
| **最近の明細**（一覧） | **ファイルを書き出す** → 全履歴をエクスポート |
| **明細詳細** | **明細を書き出す** → 1件だけエクスポート |

### 対応フォーマット

| 形式 | 用途 |
|------|------|
| **PDF** | 印刷・共有に最適（どのビューアでも開ける） |
| **CSV** | Excel / Googleスプレッドシート / Numbers |
| **Word (.doc)** | Microsoft Word / Googleドキュメント / Pages |
| **テキスト (.txt)** | メール・LINEなどにそのまま送信 |

形式を選ぶと、端末の共有シートが開き、メール・チャット・クラウドに送れます。
